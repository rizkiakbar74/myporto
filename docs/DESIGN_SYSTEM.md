# DESIGN_SYSTEM — Portfolio Cinematic Bahasa Indonesia

## 1. Arah Visual
Desain diarahkan ke portfolio cinematic modern seperti referensi video:

- Dark mode premium.
- Full-width layout.
- Hero dramatis.
- Gradient glow.
- Glass cards.
- Animasi scroll halus.
- Typography besar.
- Project showcase visual.

## 2. Warna

### Background
- Primary black: `#05070D`
- Deep navy: `#080E1A`
- Card dark: `#0D1324`

### Accent
- Cyan: `#22D3EE`
- Blue: `#3B82F6`
- Violet: `#8B5CF6`
- Soft red glow: `#EF4444`

### Text
- Main text: `#F8FAFC`
- Secondary text: `#CBD5E1`
- Muted text: `#64748B`

### Border
- Subtle border: `rgba(148, 163, 184, 0.18)`
- Active border: `rgba(34, 211, 238, 0.45)`

## 3. Typography
Rekomendasi font:
- Heading: Inter / Sora / Space Grotesk
- Body: Inter

Skala:
- Hero title desktop: 64–96px
- Hero title mobile: 42–56px
- Section title desktop: 40–56px
- Section title mobile: 32–40px
- Body: 16–18px

## 4. Layout

### Container
- Max width: 1200px
- Padding desktop: 32px
- Padding mobile: 20px

### Section spacing
- Desktop: 120px vertical
- Tablet: 96px vertical
- Mobile: 72px vertical

## 5. Komponen Visual

### Navbar
- Sticky top.
- Semi-transparent dark blur.
- Border bawah subtle.
- Mobile menu slide/dropdown.

### Hero
- Min-height: 100vh.
- Background radial gradient.
- Floating glow orbs.
- Large headline.
- CTA buttons.
- Optional portrait/mockup visual.

### Card
- Border radius: 24px.
- Background: `rgba(15, 23, 42, 0.65)`
- Backdrop blur.
- Border subtle.
- Hover: lift 4–8px + glow.

### Project Card
- Image ratio: 16:10 atau 4:3.
- Overlay gradient.
- Stack pills.
- CTA link.

### Timeline
- Vertical line.
- Dot glow per item.
- Card kanan/kiri atau stacked mobile.

## 6. Animasi
Gunakan animasi seperlunya:

- Fade up saat section masuk viewport.
- Hover lift pada cards.
- Subtle floating glow di hero.
- Text reveal ringan di headline.
- Avoid heavy 3D untuk MVP.

Durasi:
- Section reveal: 0.6–0.8s
- Hover: 0.2–0.3s
- Floating: 6–10s infinite

## 7. Responsive Rules

Mobile wajib:
- Tidak ada horizontal overflow.
- Hero text tidak terlalu besar.
- CTA buttons stack vertical jika sempit.
- Cards 1 kolom.
- Navbar mobile menu jelas.
- Project image tetap proporsional.

## 8. Style Bahasa
Tone copywriting:
- Profesional.
- Percaya diri.
- Singkat.
- Tidak lebay.
- Tidak terlalu formal seperti surat lamaran.

Contoh gaya:
> Saya membangun website yang rapi, responsif, dan mudah digunakan — dengan fokus pada struktur, tampilan, dan pengalaman pengguna.

Hindari:
> Saya yakin bisa berkontribusi di perusahaan dengan kemampuan yang saya miliki.

Karena kalimat itu terlalu umum dan terlalu terasa seperti CV.
